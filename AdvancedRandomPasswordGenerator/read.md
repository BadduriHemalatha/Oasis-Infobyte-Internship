This project is an Advanced Password Generator built using Python and Tkinter. It provides a user-friendly graphical interface to generate strong and secure passwords based on customizable user preferences.
The application ensures password strength by enforcing security rules and allows easy copying of generated passwords using clipboard integration.
🚀 Features
🔐 Strong Password Generation
Generates highly secure and random passwords
🎛 Customizable Options
Include/exclude:
Lowercase letters (a-z)
Uppercase letters (A-Z)
Numbers (0-9)
Symbols (!@#$...)
⚙️ User Input Validation
Ensures valid password length
Prevents invalid inputs
🛡 Security Rules Enforcement
Guarantees at least one character from each selected category
✂️ Character Exclusion
Users can exclude specific characters (e.g., l, 1, O, 0)
📋 Clipboard Integration
One-click copy to clipboard using pyperclip
🖥 GUI Interface
Simple and intuitive design using Tkinter
🛠 Technologies Used
Python 3
Tkinter (for GUI)
random & string modules (for password generation)
pyperclip (for clipboard functionality)
Installation

Install Required Library
Open Command Prompt and run:
py -m pip install pyperclip
🧠 How It Works
User enters desired password length
Selects character types (lowercase, uppercase, numbers, symbols)
Optionally excludes unwanted characters
The program:
Builds a character pool
Ensures at least one character from each selected type
Randomizes and generates the password
Password is displayed and can be copied instantly
Output
GUI window opens
User inputs preferences
Secure password is generated and displayed
