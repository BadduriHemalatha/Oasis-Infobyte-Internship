BMI Calculator:
Body Mass Index (BMI) Calculator built using Python. This program takes user input for weight and height, calculates BMI, and categorizes it into standard health ranges.
Features:
Calculates BMI using standard formula
Categorizes BMI into:
Underweight
Normal weight
Overweight
Obese
Handles invalid inputs gracefully
Simple and beginner-friendly Python code
